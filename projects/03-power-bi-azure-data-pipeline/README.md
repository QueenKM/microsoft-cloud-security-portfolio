# Project 03: Power BI + Azure Data Pipeline

## Goal

Build an end-to-end analytics solution that transforms operational or security data into a Power BI dashboard with clear KPIs and a clean semantic model.

## Why This Project Matters

This project adds strong analytics credibility and shows that you can turn raw data into useful decisions, not just collect logs.

## Suggested Theme

Use security operations and governance data from the wider portfolio, such as:

- sign-in activity
- incidents by severity
- device compliance trends
- privileged role assignments
- Teams guest access counts

## Core Scope

- dataset ingestion from CSV, SQL, or Azure source
- data cleaning and transformation
- star schema modeling
- DAX measures
- Power BI dashboard with KPIs

## Deliverables

- source dataset
- transformation notes
- star schema diagram
- DAX measures list
- dashboard screenshots
- insights summary

## Build Phases

1. Select a dataset and define the business questions.
2. Model facts and dimensions in a star schema.
3. Build measures and KPIs in Power BI.
4. Design a clean dashboard for executive and analyst views.
5. Document refresh flow and lessons learned.

## Evidence To Capture

- schema diagram
- Power Query transformations
- DAX measure examples
- dashboard views
- one-page insight summary
